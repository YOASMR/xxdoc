# 一、导入订阅

*部分服务商提供快速的一键导入到客户端，免去了复制 -> 打开软件 -> 添加的过程

因此我们这里只讲手动导入的教程，因为一键导入只是少了一步打开软件然后点添加的步骤。

# 二、开始启用

点击配置

点击新配置 添加配置

点击从URL导入

填写配置信息

名称自己自定义

URL是加速服务商给你提供的订阅链接

你购买加速服务后，在网站能够看到快速复制配置的按钮或链接

自动更新 建议填写：1440

自动更新这个比较重要，因为服务商无法保障每一个节点都可用 因此会经常更新节点信息，如果你本地订阅不更新的话，你无法获得最新的节点连接信息，时间稍微一场，很可能你的节点都连不了了。

填写完后点击右上角保存，保存需要2-3秒，等一下就好。

然后返回，如果这是你的第一个配置的话，需要先选中配置，在配置页面点击一下选中就行。

继续返回到软件主页。

点击第一个方块按钮 点此启动 变成蓝色

已停止 变为 运行中

此时，你基本就可以正常使用了。

服务商一般提供了全球多个地区的节点

点击 代理 进入选择 你需要的地区节点
