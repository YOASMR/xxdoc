# 全局与规则 （必看）

如果你是选择了全局，则所有的网络流量都会走代理，但是国内的服务也会走代理，所以可能会变慢，比如你打开百度，会变慢变卡，看腾讯视频也会变卡。

如果是选择规则的节点，则是自动识别，比如你访问Google，就会走代理，访问百度或者国内的服务还是走系统原来的国内线路，这是使用比较舒服的操作。

普通用户建议选择规则下面的节点就行。

# 系统状态栏快速修改

通过windows右下角的小猫咪图标右键，可以快速选择，比如通过点击System proxy 开启或关闭代理。
